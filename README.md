Karvo 🔧⚡
A local services marketplace that connects customers with verified skilled workers — tutors, tailors, electricians, plumbers, AC technicians, carpenters, painters and mehndi artists.

Live demo: (add your GitHub Pages link here)

Features
Smart search by service, keyword, or area (understands "AC repair", "leak", "tuition" etc.)
8 service categories with filterable pro listings, ratings & reviews
Pro profiles with rating breakdowns and availability badges
Booking requests and pro registration delivered via WhatsApp
Job cost calculator and transparent price guide
Packages, seasonal offers, FAQ, blog tips & trust/safety page
English/Urdu toggle and light/dark theme
Fully responsive — works on mobile and desktop
Tech
Single-file build: pure HTML, CSS and vanilla JavaScript. No frameworks, no build step, no backend — bookings route through WhatsApp.

Running it
Download karvo-marketplace.html and open it in any browser. That's it.
